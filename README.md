# agrovision-futuro-sustentavel
AgroVision é um projeto educacional desenvolvido para o Agrinho 2026 com o objetivo de conscientizar sobre a importância da sustentabilidade no agronegócio e mostrar como as escolhas atuais influenciam o futuro da produção de alimentos, do meio ambiente e da sociedade.

O site apresenta informações sobre o agronegócio moderno, tecnologias sustentáveis e soluções inovadoras que contribuem para uma produção mais eficiente e ambientalmente responsável.

Funcionalidades
Apresentação da importância do agronegócio para o Brasil;
Exibição de tecnologias sustentáveis aplicadas ao campo;
Área interativa para explorar escolhas sustentáveis e seus benefícios;
Contadores dinâmicos que simulam impactos positivos para o meio ambiente;
Layout responsivo para computadores, tablets e smartphones;
Interface moderna com imagens relacionadas à agricultura e sustentabilidade.
Tecnologias Utilizadas
HTML5
CSS3
JavaScript
Objetivo

Demonstrar como a inovação, a tecnologia e as práticas sustentáveis podem contribuir para a construção de um futuro mais produtivo, equilibrado e consciente.
